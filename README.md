# 🎨 SaaS Landing Page UI Clone

A responsive **SaaS product landing page UI clone** built using **HTML, CSS, and JavaScript**.
This project was created as a frontend practice project while learning web development fundamentals.

---

## 🚀 Project Overview

This project is a **frontend-only UI clone** inspired by modern SaaS product websites.
It focuses on layout design, responsiveness, and clean UI structure without any backend functionality.

---

## ✨ Features


* Modern UI layout inspired by SaaS platforms
* Clean and structured HTML5 markup
* Custom CSS styling (no frameworks used)
* Basic JavaScript interactions
* Separate pages:

  * Home (index.html)
  * Contact page
* Optimized image and asset structure

---
